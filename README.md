# Analytics Dashboard

A React-based Analytics Dashboard built for learning and practicing modern frontend development concepts.

This project helped me understand how large dashboards are structured using reusable components, charts, responsive layouts, and state management techniques.

## 🚀 Features

* Responsive Dashboard Layout
* Sidebar & Navbar
* KPI Cards
* Charts & Analytics Visualizations
* Recent Activity Feed
* Team & Settings Cards
* Dark / Light Theme Toggle
* Theme Persistence using localStorage
* Smooth Animations

## 🛠️ Tech Stack

* React.js
* Vite
* Tailwind CSS
* Framer Motion
* Recharts
* Lucide React

## 📚 What I Learned

Through this project, I improved my understanding of:

* Component-Based Architecture
* Props & Data Flow
* Responsive UI Design
* Chart Integration with Recharts
* Framer Motion Animations
* Theme Management
* React Context API
* localStorage Persistence
* Reusable Components
